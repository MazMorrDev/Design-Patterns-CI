# Design-Patterns-CI

## About

This repository contains implementations of various design patterns categorized by:

- Creational Patterns
- Structural Patterns  
- Behavioral Patterns

## Prerequisites

- Git installed on your system
- Basic knowledge of design patterns
- Programming languages of your choice

## Quick Start

1. Ensure to contact me to be a collaborator
2. Create a folder in your local computer
3. Inside The folder create a new git project using

    ```bash
    git init
    ```

4. Then just pull the repo and start coding those patterns :D

    ```bash
    git pull https://github.com/MazMorrDev/Design-Patterns-CI.git
    ```

## Collaborations

1. You must ensure that you created a folder for the design patterns implementation like this example

    ```text
    Creational/YourDesignPattern/YourProgrammingLanguage/YourImplementation
    ```

2. There must be comments explainig how your pattern works (if you dont know english it can be in spanish)
