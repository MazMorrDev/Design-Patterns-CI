# Design-Patterns-CI

## Acerca del Proyecto

Este repositorio contiene implementaciones de varios patrones de diseño categorizados en:

- Patrones Creacionales
- Patrones Estructurales  
- Patrones de Comportamiento

## Prerrequisitos

- Git instalado en tu sistema
- Conocimiento básico de patrones de diseño
- Lenguajes de programación de tu elección

## Inicio Rápido

1. Asegúrate de contactarme para ser colaborador
2. Crea una carpeta en tu computadora local
3. Dentro de la carpeta, crea un nuevo proyecto git usando:

    ```bash
    git init
    ```

4. Luego simplemente haz pull del repositorio y comienza a programar esos patrones :D

    ```bash
    git pull https://github.com/MazMorrDev/Design-Patterns-CI.git
    ```

## Colaboraciones

1. Debes asegurarte de crear una carpeta para la implementación de patrones de diseño como en este ejemplo:

    ```text
    Creational/TuPatronDeDiseño/TuLenguajeDeProgramacion/TuImplementacion
    ```

2. Debe haber comentarios explicando cómo funciona tu patrón (si no sabes inglés, puede ser en español)

3. Tus commits deben usar conventional commits, si no estás familiarizados con ellos puedes visitar: [Conventional Commits Page](https://www.conventionalcommits.org/en/v1.0.0/)
